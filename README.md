# RxJs Boilerplate

RxJs boilerplate to build reactive application with Webpack 4

## Requirements
- Node 8+
- npm 5+

## Usage

### Installation
Install dependencies:
```
npm i
```

Run application in `production` mode:
```
npm start
```

Run application in `development` mode with webpack-dev-server with file watch
```
npm build
```
Visit: `http://localhost:8080`

### Dependencies
- rxjs v6
- webpack v4
- webpack CLI
- webpack dev server
- babel v7
- node-sass
